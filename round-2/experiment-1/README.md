## Full Summary

Completed experiment comparing Thermodynamic Entropy Calibration (TEC) vs Temperature Scaling (TS) on 5 text classification datasets (SST-2, QNLI, AG News, MNLI, DBpedia) using pre-trained transformers. Methods: (1) Uncalibrated baseline, (2) Temperature Scaling, (3) TEC with per-sample temperature based on entropy + margin. Metrics: ECE, Brier score, NLL, Accuracy with bootstrap CI (200 samples). Heterogeneous analysis on easy/hard splits by decision margin. Results: TEC outperformed TS on 2/5 datasets (QNLI, DBpedia), TS outperformed TEC on 3/5 datasets. Both methods significantly improved calibration over uncalibrated baseline. Output saved to method_out.json in exp_gen_sol_out schema format with per-example predictions.
